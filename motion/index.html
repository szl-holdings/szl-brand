Doctrine made visible.

We built an AI agent body — brain, heart, hands, and wires — and every heartbeat produces a sha256 receipt you can verify. These animations aren't decoration. Each one is a different entry point into the same audited system. No hallucinations, no bandaids; the hash you see is the hash on disk.

FIG 1.1 · HEART

Beating heart

The gold YUYAY v3 heart contracts on a 1.2-second beat — 40% systolic snap, 60% diastolic ease. On every beat a truncated sha256 falls into the YAWAR append-only bus and the bus flashes gold. The hash is injected at build time from git HEAD.

Recommended surface: a11oy, landing pages, LinkedIn hero.

FIG 1.2 · OUROBOROS

Receipt rain

A counted stream of sha256 hashes falls into the gold YAWAR bus. The bus lights up where each receipt lands — the audit loop in motion. Six lanes, 3.2-second fall, staggered every 550 ms. HUKLLA watches from the right.

Recommended surface: ouroboros, counsel, Substack hero.

FIG 1.3 · SENTRA

HUKLLA tripwire ring

Ten deterministic tripwires (T01..T10) surround the DEADMAN core. Each tripwire pulses rust in sequence on a 6-second sweep — security on patrol, not noise. The outer dashed ring rotates at 30s per revolution. Failure on any tripwire equals freeze.

Recommended surface: sentra, security dashboards, alert states.

Constraints these animations honor

  • Pure inline SVG + CSS @keyframes — no JavaScript, no external libraries
  • prefers-reduced-motion fallback baked in: each animation has a static state
  • Palette locked to SZL tokens: #F5F1E8 cream, #1A1A1A ink, #B08940 gold
  • Fonts are system-safe — no proprietary Tiempos / Styrene / OpenAI Sans embeds
  • GitHub README-deployable as <img src="heart-beat.svg">
  • The hash shown is the hash on disk — no fake particle counts, no decorative noise
