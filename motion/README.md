# SZL Motion Library v1

Doctrine made visible. Three zero-dependency SVG+CSS animations.
Each loops with `prefers-reduced-motion` fallback. No JS. No external libs.

| Animation | Role | Period | sha256 anchor |
|---|---|---|---|
| `heart-beat.svg` | YUYAY heart pulse with receipt drop | 1.2s | `d94c683e` |
| `receipt-rain.svg` | YAWAR bus receipt stream | 3.2s | injected daily |
| `tripwire-ring.svg` | HUKLLA observer ring sweep | 6.0s | per-tripwire |

**Open `index.html` to view all three side-by-side.**
